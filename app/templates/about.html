About BookDrop

  • What is BookDrop?

    BookDrop is an easy way to transfer books to your Kindle using a Dropbox folder.

  • How do I use BookDrop?

    It's simple! Just register here, and start adding books to your Dropbox/Apps/book-drop folder. Any new books you add will get sent to your Kindle.

  • Can I use BookDrop to share books with friends?

    Yup, just follow these steps:

    1. Create a folder for books in your Dropbox folder.
    2. Go to the Dropbox website, and share the folder with a friend.
    3. Using a file manager like "Finder" on Mac OS, create a shortcut to the shared folder inside Dropbox/Apps/book-drop.
    4. Have your friend create a similar shortcut to their copy of the shared folder. Now you'll both receive any books that get added to the folder!

  • What kind of reading devices can I use BookDrop with?

    You can use BookDrop with any Kindle e-reader or mobile app.

  • What kind of files can I send using BookDrop?

    You can send any files supported by Amazon. BookDrop also supports any .epub, .cbr, and .cbz files, which BookDrop will convert to a Kindle-supported format (the quality of these conversions may vary from file to file). In total, these are:

    • .epub, .mobi, .pdf, .azw
    • .cbr, .cbz
    • .txt, .rtf, .doc, .docx, .htm, .html
    • .gif, .png, .bmp, .jpg, .jpeg

    If there's a file format that you don't see here, send us an email, and we'll see what we can do!

  • Help! I've added books to my BookDrop folder, but I haven't received them on my Kindle for some reason.

    First things first, make sure your Kindle is connected to WiFi, and wait a few minutes. It takes time to convert and send your book.

    If you still haven't received the file, make sure that it's of one of the supported filetypes listed above and below 25MB. Also, for legal reasons, BookDrop can't convert files under DRM.

    If all else fails, send us an email detailing what happened.

  • Can I delete books from my Kindle using BookDrop?

    Unfortunately, no. We'll have to wait for a Kindle API :(

  • I have some books that I've already transferred to my Kindle. If I add them to my BookDrop folder, will they get added to my Kindle again?

    Unfortunately, yes. This is because BookDrop has no way of knowing what files are already on your Kindle.

    There is a way around this, though. BookDrop will only send files of a certain type, so if you rename a file with an unsupported file extension (e.g. ".kbox"), it won't get sent to your Kindle. Make sure to do this before you add the file to your BookDrop folder.

  • Can I register for BookDrop on mobile?

    Nope :( Perhaps in a galaxy far far away...

  • Can I connect multiple Dropbox accounts to the same Kindle?

    Yup, just make sure to register your different Dropbox accounts with the same Kindle. You can also connect multiple Kindles to the same Dropbox account, as long as they all belong to the same Amazon account.

  • How does BookDrop work under the hood?

    BookDrop works on top of Amazon's existing Kindle Personal Documents Service. Through this service, you can give certain email addresses permission to send documents to your Kindle, free of charge. Your Kindle can then download these documents wirelessly.

    Whenever you add a book to Dropbox/Apps/book-drop, BookDrop is notified by the Dropbox webhooks service. BookDrop then downloads the book through the Dropbox Core API and emails it to your Kindle.

  • What does the "Activate BookDrop" bookmarklet do? Should I be worried about running this JavaScript in my browser?

    The bookmarklet does a couple things. First, it scrapes the Amazon Manage Your Content and Devices page for possible BookDrop devices and displays them in a modal for you to pick from. Second, the bookmarklet sends a request to Amazon to allow a BookDrop email address to send your Kindle devices personal documents. If all goes well, the bookmarklet notifies BookDrop that your account is all set.

    Here on GitHub.
