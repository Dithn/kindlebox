About Kindlebox

  • What is Kindlebox?

    Kindlebox is an easy way to transfer books to your Kindle using a Dropbox folder.

  • How do I use Kindlebox?

    It's simple! Just register here, and start adding books to your Dropbox/Apps/kindle-box folder. Any new books you add will get sent to your Kindle.

  • Can I use Kindlebox to share books with friends?

    Yup, just follow these steps:

    1. Create a folder for books in your Dropbox folder.
    2. Go to the Dropbox website, and share the folder with a friend.
    3. Using a file manager like "Finder" on Mac OS, create a shortcut to the shared folder inside Dropbox/Apps/kindle-box.
    4. Have your friend create a similar shortcut to their copy of the shared folder. Now you'll both receive any books that get added to the folder!

  • What kind of reading devices can I use Kindlebox with?

    You can use Kindlebox with any Kindle e-reader or mobile app.

  • What kind of files can I send using Kindlebox?

    You can send any files supported by Amazon, in addition to .epub, which Kindlebox will convert to a Kindle-supported format. In total, these are:

    • .epub, .mobi, .pdf, .azw
    • .txt, .rtf, .doc, .docx, .htm, .html
    • .gif, .png, .bmp, .jpg, .jpeg

  • Help! I've added books to my Kindlebox folder, but I haven't received them on my Kindle for some reason.

    First things first, make sure your Kindle is connected to WiFi, and wait a few minutes. Because Kindlebox depends on Internet connections to both Dropbox and Amazon, there could be a slight delay in receiving your books.

    If you still haven't received the file, make sure that it's of one of the supported filetypes listed above and that it's below 15MB. If all else fails, send us an email detailing what happened.

  • Can I delete books from my Kindle using Kindlebox?

    Unfortunately, no. We'll have to wait for a Kindle API :(

  • I have some books that I've already transferred to my Kindle. If I add them to my Kindlebox folder, will they get added to my Kindle again?

    Unfortunately, yes. This is because Kindlebox has no way of knowing what files are already on your Kindle.

    There is a way around this, though. Kindlebox will only send files of a certain type, so if you rename a file with an unsupported file extension (e.g. ".kbox"), it won't get sent to your Kindle. Make sure to do this before you add the file to your Kindlebox folder.

  • Can I register for Kindlebox on mobile?

    Nope :( Perhaps in a galaxy far far away...

  • Can I connect multiple Dropbox accounts to the same Kindle?

    Yup, just make sure to register your different Dropbox accounts with the same Kindle. You can also connect multiple Kindles to the same Dropbox account, as long as they all belong to the same Amazon account.

  • How does Kindlebox work under the hood?

    Kindlebox works on top of Amazon's existing Kindle Personal Documents Service. Through this service, you can give certain email addresses permission to send documents to your Kindle, free of charge. Your Kindle can then download these documents wirelessly.

    Whenever you add a book to Dropbox/Apps/kindle-box, Kindlebox is notified by the Dropbox webhooks service. Kindlebox then downloads the book through the Dropbox Core API and emails it to your Kindle.

  • What does the "Activate Kindlebox" bookmarklet do? Should I be worried about running this JavaScript in my browser?

    The bookmarklet does a couple things. First, it scrapes the Amazon Manage Your Content and Devices page for possible Kindlebox devices and displays them in a modal for you to pick from. Second, the bookmarklet sends a request to Amazon to allow a Kindlebox email address to send your Kindle devices personal documents. If all goes well, the bookmarklet notifies Kindlebox that your account is all set.
